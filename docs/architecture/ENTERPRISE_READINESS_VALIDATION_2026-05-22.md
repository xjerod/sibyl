   cluster backup-drill receipt, audit export sample, release SBOM artifact, Cosign signature
   receipt, and lockfile/package diff for Authlib, PyJWT, and argon2-cffi.

Until those artifacts are captured, the code-side implementation is locally validated but the full
enterprise acceptance gate is not fully closed.
